# University Portal 🎓

A multi-role university management system 
built with Flask and MySQL.

## Features
- Separate login for Students and Teachers
- Student registration with Roll No
- Teacher registration with ID No
- Student dashboard with Subjects and Marks
- Teacher dashboard to upload marks for students
- Subject selection system (choose 3 subjects)
- Students can view their marks per subject
- Profile photo upload and edit
- Form validation with error messages

## Tech Stack
- Backend: Python, Flask
- Database: MySQL
- Frontend: HTML, CSS
- Deployment: Railway

## Live Demo
🔗 https://university-portal.up.railway.app/

## How It Works
- **Student** logs in → selects subjects → 
  views marks uploaded by teacher
- **Teacher** logs in → sees all students → 
  uploads marks per subject
